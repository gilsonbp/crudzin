# CRUDZIN Flask

## Projeto com um exemplo de CRUD Flask
> Python 3.7.x or higher is required

### Create virtualenv
```sh
python -m venv .venv
```

### Install dependencies
```sh
pip install -r requerements.txt
```

### Migrations:
```sh
flask db init
flask db migrate
flask db update
```

### Run server:
```sh
flask run
```
